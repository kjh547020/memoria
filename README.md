## 📢 홍보용 줄거리

**"당신의 모든 기억은 데이터이며, 모든 데이터는 거래된다. 아버지의 죽음에 얽힌 진실을 쫓아, 도현은 도시를 장악한 기억거래소 MEMORIA에 맞서거나, 굴복하거나, 아니면 파멸해야 한다."**

**2048년. 기억은 더 이상 사적인 영역이 아니다.
기억추출 기술의 상용화 이후, 기억은 상품이 되고 경험은 화폐가 되었다.**

기억거래소에서 일하는 기억추출관 도현.
그는 2년 전 실종된 아버지의 흔적을 좇지만,
조각난 기억과 지워진 진실이 그의 길을 가로막는다.

조작된 CCTV, 손상된 기억카드, 거래소의 압박,
그리고 어머니의 침묵.

플레이어는 변수와 선택을 통해
기억의 미궁 속에서 진실을 복원하거나,
혹은 자신을 잃어버릴 수도 있다.

## 👥 인물 소개

| 인물 | 역할 및 서사적 기능 |
| :--- | :--- |
| **도현** | **주인공**. 아버지에 대한 진실을 추적하는 기억추출관. 구멍난 기억을 채워가며 진실을 찾아야만 한다. |
| **어머니** | 도현의 유일한 가족. 사건 이후 도현과의 관계가 악화되며, 그녀가 도현에게 미치는 영향은 매우 크다. |
| **기억거래소** | 도시의 기억과 정보를 독점하는 거대 기업. |
| **김창수** | 거래소와 경찰 사이의 경계인. 도현의 조력자가 되기도 한다. |
